Kconnect-search
===============

Kconnect-search engine front-end

[![Dependency Status](https://gemnasium.com/badges/github.com/healthonnet/Kconnect-search.svg)](https://gemnasium.com/github.com/healthonnet/Kconnect-search)

## Install

```
$ npm install
$ npm test
$ gulp
```

## Browser reload

```
$ gulp serve
```
